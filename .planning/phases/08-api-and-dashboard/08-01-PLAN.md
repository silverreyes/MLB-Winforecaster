from starlette.responses import FileResponse
from pathlib import Path

class SPAStaticFiles(StaticFiles):
    async def get_response(self, path: str, scope):
        try:
            return await super().get_response(path, scope)
        except Exception:
            return FileResponse(Path(self.directory) / "index.html")

Define lifespan:
```python
@asynccontextmanager
async def lifespan(app: FastAPI):
    artifacts = load_all_artifacts()  # Raises FileNotFoundError if missing (API-06)
    pool = get_pool(min_size=2, max_size=5)
    app.state.artifacts = artifacts
    app.state.pool = pool
    yield
    pool.close()
```

**1. Create api/routes/predictions.py** with three endpoints on an `APIRouter(tags=["predictions"])`:

**GET /predictions/today** (API-01):
- Use `def` (NOT `async def`) -- sync handler runs in FastAPI thread pool
- Access pool via `request.app.state.pool`
- Query: `SELECT * FROM predictions WHERE game_date = CURRENT_DATE AND is_latest = TRUE ORDER BY home_team, away_team`
- Use `psycopg.rows.dict_row` cursor factory
- For each row, compute `ensemble_prob = round((lr_prob + rf_prob + xgb_prob) / 3, 4)` if all three are non-None, else None
- For each row, compute `edge_magnitude`: if `kalshi_yes_price` is not None and `ensemble_prob` is not None, compute `round((ensemble_prob - kalshi_yes_price) * 100, 1)` (in percentage points); else None
- Map each row to `PredictionResponse`, collecting into a list
- Compute `latest_prediction_at` as `max(row["created_at"] for row in rows)` if rows exist, else None
- Return `TodayResponse(predictions=[...], latest_prediction_at=..., generated_at=datetime.now(timezone.utc))`

**GET /predictions/latest-timestamp** (API-03):
- Query: `SELECT MAX(created_at) as latest FROM predictions WHERE game_date = CURRENT_DATE AND is_latest = TRUE`
- Return `LatestTimestampResponse(timestamp=result)`
- ROUTE ORDER IS CRITICAL: This decorator (`@router.get("/predictions/latest-timestamp")`) MUST appear before `@router.get("/predictions/{date}")` in the file. If `/{date}` registers first, FastAPI captures the literal string "latest-timestamp" as a date parameter and returns a 422. Add this comment directly above the decorator in the source file: `# DO NOT REORDER: /latest-timestamp must precede /{date} to prevent path parameter capture`

**GET /predictions/{date}** (API-02):
- Path parameter `date: str` with format validation (YYYY-MM-DD)
- Identical logic to /today but with `WHERE game_date = %(date)s` instead of `CURRENT_DATE`
- Return same `TodayResponse` shape
- This route must be the LAST route defined in the router (after /today and /latest-timestamp)
