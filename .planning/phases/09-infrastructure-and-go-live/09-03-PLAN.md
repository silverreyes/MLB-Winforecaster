Complete deployment stack: Docker Compose (api + worker + db), Nginx reverse proxy, SSL certificate, Postgres volume persistence, backup cron. All created in Plans 01 and 02.
  </what-built>
  <how-to-verify>
Deploy the stack to the Hostinger KVM 2 VPS and verify each requirement:

**Pre-Deploy (on VPS):**
1. Clone/pull the repo to the VPS
2. Copy model artifacts to models/artifacts/ on the VPS (scp from local)
3. Create .env from .env.example: `cp .env.example .env && nano .env` (set POSTGRES_PASSWORD)
4. Run memory audit: `bash scripts/memory_audit.sh` -- must print "DEPLOY GATE: PASS"

**Deploy Docker Stack:**
5. Build and start: `docker compose up -d --build`
6. Wait for healthy: `docker compose ps` -- all 3 services should be "running" or "healthy"
7. Verify memory limits: `docker stats --no-stream` -- api < 512MB, worker < 1GB, db < 512MB
8. Verify API responds: `curl http://localhost:8082/api/v1/health` -- should return JSON with status

**Setup Nginx + SSL:**
9. Copy nginx config: `sudo cp nginx/mlbforecaster.conf /etc/nginx/sites-available/mlbforecaster.silverreyes.net`
10. Enable site: `sudo ln -s /etc/nginx/sites-available/mlbforecaster.silverreyes.net /etc/nginx/sites-enabled/`
11. Validate: `sudo nginx -t` -- must print "syntax is ok"
12. Reload: `sudo systemctl reload nginx`
13. Test HTTP: `curl -I http://mlbforecaster.silverreyes.net` -- should get 200 OK
14. Issue SSL cert: `sudo certbot --nginx -d mlbforecaster.silverreyes.net`
15. Test HTTPS: `curl -I https://mlbforecaster.silverreyes.net` -- should get 200 OK
16. Renewal dry-run: `sudo certbot renew --dry-run` -- must succeed

**Verify Postgres Persistence (INFRA-04):**
17. Stop stack: `docker compose stop`
18. Start stack: `docker compose start`
19. Verify data: `docker exec mlb-winforecaster-db-1 psql -U mlb -d mlb_forecaster -c "SELECT count(*) FROM predictions;"` -- count should be same as before stop

**Setup Backup Cron:**
20. Test backup: `bash scripts/backup_postgres.sh`
21. Verify backup file: `ls -la /opt/backups/mlb/` -- should show timestamped .sql.gz file
22. Install cron: `(sudo crontab -l 2>/dev/null; echo "0 3 * * * /opt/mlb-winforecaster/scripts/backup_postgres.sh >> /var/log/mlb_backup.log 2>&1") | sudo crontab -` then verify: `sudo crontab -l`

**Final Verification:**
23. Visit https://mlbforecaster.silverreyes.net in browser -- should show dashboard
24. Check docker stats: total memory usage should leave >1GB headroom on 8GB VPS
