```python
#!/usr/bin/env python
"""One-time seed script: backfill game_logs from MLB Stats API.

Fetches all Final regular-season games for the specified seasons and inserts
them into the game_logs Postgres table. Idempotent via ON CONFLICT DO NOTHING.

Usage:
    python scripts/seed_game_logs.py                    # Default: 2025 2026
    python scripts/seed_game_logs.py --seasons 2025     # Only 2025
    docker compose exec worker python scripts/seed_game_logs.py
"""
```

Implementation details:
1. **Imports:** argparse, logging, statsapi, from src.pipeline.db import get_pool, apply_schema, batch_insert_game_logs, from src.data.team_mappings import normalize_team, from src.data.mlb_schedule import SEASON_DATES
2. **CLI:** `argparse.ArgumentParser` with `--seasons` accepting nargs='+' of ints, default `[2025, 2026]`
3. **Main flow:**
   - Call `get_pool()` and `apply_schema(pool)` to ensure migration_002 has run
   - For each season in args.seasons:
     a. Look up `(start_date, end_date)` from `SEASON_DATES[season]`
     b. Call `statsapi.schedule(start_date=start_date, end_date=end_date, sportId=1)`
     c. Filter: `g.get("game_type") == "R" and g.get("status") == "Final"`
     d. Normalize each game dict using the `_SKIP_NORMALIZE = {"", "Tie"}` pattern:
        - `game_id`: `str(g["game_id"])`
        - `game_date`: `g["game_date"]`
        - `home_team`: `normalize_team(g["home_name"])`
        - `away_team`: `normalize_team(g["away_name"])`
        - `home_score`: `g["home_score"]`
        - `away_score`: `g["away_score"]`
        - `winning_team`: `normalize_team(g["winning_team"]) if g.get("winning_team") not in _SKIP_NORMALIZE else g.get("winning_team", "")`
        - `losing_team`: same pattern as winning_team
        - `home_probable_pitcher`: `g.get("home_probable_pitcher") or None`
        - `away_probable_pitcher`: `g.get("away_probable_pitcher") or None`
        - `season`: `season` (the loop variable)
     e. Call `batch_insert_game_logs(pool, normalized_games)`
     f. Log: `"Season {season}: fetched {len(final_games)} Final games, inserted {inserted} new rows, skipped {len(final_games) - inserted} duplicates"`
4. **Error handling:** Wrap normalize_team calls in try/except ValueError, log and skip individual games that fail normalization
5. **Logging:** `logging.basicConfig(level=logging.INFO, format="%(asctime)s [%(name)s] %(levelname)s: %(message)s")`
6. At the end, log total across all seasons: `"Seed complete: {total_inserted} total new rows across {len(seasons)} seasons"`
7. Close pool at the end: `pool.close()`
